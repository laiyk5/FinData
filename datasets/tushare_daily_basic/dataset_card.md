# Dataset Card: Tushare Daily Basic

## Summary

`tushare_daily_basic` stores A-share daily valuation, share-capital, turnover,
and market-cap indicators from the Tushare Pro `daily_basic` API.

It complements `tushare_daily`: `tushare_daily` stores OHLCV price bars, while
`tushare_daily_basic` stores derived daily market indicators such as turnover
rate, PE/PB/PS, dividend yield, share counts, and market value.

## Intended Use

This dataset is intended for cross-sectional screening, factor research,
portfolio diagnostics, and daily market dashboards. Consumers should treat
valuation fields as provider-calculated daily indicators, not audited financial
statement values.

## Source

Provider: Tushare Pro

API: `daily_basic`

Documentation: https://tushare.pro/document/2?doc_id=32

According to the provider documentation, `daily_basic` is updated on trading
days around 15:00-17:00 China time and returns up to 6000 rows per request. The
documented request parameters include `ts_code`, `trade_date`, `start_date`, and
`end_date`; either `ts_code` or `trade_date` is required.

## Grain

One row represents one Tushare security code on one trading date.

Expected primary key:

```text
ts_code, trade_date
```

## Known Missingness

Expected missingness categories include:

- market holidays and non-trading days
- suspended securities
- newly listed securities outside the requested date range
- provider-side unavailable metrics
- valuation fields that are blank because they are not meaningful, such as PE
  for loss-making companies

Missing rows should be classified through QA and accepted missingness records,
not synthesized.

## Storage

CSV is the v1 published storage format. Raw Tushare responses are kept in run
sandboxes for replay and debugging.

## Validation Expectations

Minimum checks include:

- required fields exist
- primary keys are unique
- `trade_date` is valid `YYYYMMDD`
- non-valuation numeric fields parse as decimals
- share and market-value fields are non-negative
- `float_share <= total_share`
- `free_share <= float_share`
- missing rows are documented or accepted

PE fields may be blank when unavailable or not meaningful.
